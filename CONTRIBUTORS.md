## Special Thanks

The ISVS project originated as an appendix in the [OWASP Application Security Verification Standard (ASVS)](https://owasp.org/www-project-application-security-verification-standard/). We extend our gratitude to the ASVS team for their pioneering work that laid the foundation for this standard.

### Referenced Projects & Standards
We acknowledge and thank the maintainers of complementary OWASP projects:
- [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/) - Application Security Verification Standard
- [OWASP MASVS](https://owasp.org/mas) - Mobile Application Security Verification Standard
- [OWASP SCVS](https://owasp.org/www-project-software-component-verification-standard/) - Software Component Verification Standard

## How to Contribute

We welcome contributions from the community! Please see our [Contributing Guide](Contributing.md) for information on how to:
- Report issues and suggest improvements
- Submit pull requests
- Join our [Slack channel (#project-isvs)](https://owasp.slack.com/messages/project-isvs)
- Help with translations
